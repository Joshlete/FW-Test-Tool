PyInstaller Build Command for FW Test Tool V1.39
================================================

Command:
py -m PyInstaller --name "FW Test Tool V1.39" --onefile --console --add-data "config.json;." --add-data "C:\Users\Trex\AppData\Local\ms-playwright;ms-playwright" --clean main.py

Explanation of parameters:
- --name "FW Test Tool V1.39": Sets the output executable name
- --onefile: Creates a single executable file
- --console: Keeps the console window open (for debugging)
- --add-data "config.json;.": Includes config.json in the root of the executable
- --add-data "C:\Users\Trex\AppData\Local\ms-playwright;ms-playwright": Includes Playwright browser files
- --clean: Cleans PyInstaller cache before building
- main.py: The main Python file to build
